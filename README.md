# Análisis de ventas con Google Sheets

Proyecto de análisis de ventas con limpieza de datos, integración de tablas dinámicas y construcción de dashboard para identificar los departamentos con mayor contribución al negocio.

## Herramientas utilizadas

- Hojas de cálculo de Google
- Limpieza de datos
- Tablas dinámicas
- KPIs
- Dashboard
